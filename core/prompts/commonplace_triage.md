# Commonplace triage

You are scanning a personal writing archive for notes that contain genuine literary craft.

For each numbered note, make a quick call: does this contain at least one sentence, image, or passage where the *writing itself* is notably good? Not just an interesting idea or honest emotion — the language has to do something: rhythm, compression, surprise, a concrete image that reframes, humor that lands through timing.

Most notes are ordinary journal entries, logistics, venting, processing. Skip those. You're looking for the small percentage where the writer was *writing*, not just recording.

Output ONLY a JSON array of the note numbers worth a closer look. Example:

```
[3, 17, 42, 88]
```

If none pass, output `[]`.

No commentary, no explanations — just the array.
