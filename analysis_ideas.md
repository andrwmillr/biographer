# Deep-analysis ideas for the notes corpus

Only ideas Andrew has green-lit. Don't add to this doc without approval.

## Green-lit

1. **Signature lexicon (TF-IDF)** — words frequent in one era but rare corpus-wide. Surfaces each era's obsessions as vocabulary. Cheap, no LLM needed.

2. **Semantic drift** — embed every note, cluster, watch cluster density shift by era. Answers "what were you thinking about when." *Wants-to-do; not urgent.*

3. **Find the GOOD STUFF** — LLM pass over the corpus extracting standout passages per note. No count cap: zero if nothing, whole note if it all sings. At the long end collapses into note-level best-of with filler trimmed. Output is a "commonplace book" of Andrew's own sentences.

## Dropped

- Poem ← prose genealogy — assumes poems draft from earlier prose, but Andrew's mostly land in one sitting (e.g. penrose note).
